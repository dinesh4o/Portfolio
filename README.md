# Portfolio

This is a Framer static export served with Express.js.

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.

## Setup Instructions

1. Clone the repository and navigate to the project directory (if you haven't already):
   ```bash
   git clone https://github.com/dinesh4o/Portfolio.git
   cd Portfolio
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

## Running the Application

To start the server, run the following command:

```bash
npm start
```

By default, the server will run on port `3000`. You can view the application by opening your browser and navigating to [http://localhost:3000](http://localhost:3000).

## Project Structure

- `server.js` - The Express server configuration and routing.
- `package.json` - Project dependencies and scripts.
- `*.html` - The exported static pages from Framer.
